## 概要

今日はmaxとappendについて学んだ。

### 学んだこと

- max関数

max関数は、与えられた複数の値やデータ構造の中から最大値を返す組み込み関数

```
# 複数の引数を直接渡す
result = max(5, 2, 8, 1)
print(result)
8
```

- append

list.append(x)
リストの末尾に要素を一つ追加する。

```
output.append(candy + extraCandies >= max_candy)
```
のような比較演算子を使う場合直接TrueかFalseを返す
